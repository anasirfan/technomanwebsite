Online Particle Monitoring System
Remote 5104

Following the latest laser optical particle counting technology, we offer the recent variant of Remote 5104 series for hassle-free processing.

It is a convenient and cost-effective device providing the real-time data ranging sensitivity from 0.5 µm and flow rate 1.0 CFM.

Its vacuum source makes it easy and flexible to mount anywhere, where a suitable space is available. It can integrate itself into huge facility monitoring and management system.

Key Features:

0.5 - 25 µm Particle Size Range

1.0 CFM (28.3 LPM) Flow Rate

4 Channels of Simultaneous Data

RS-485 Modbus Output

Meets 21501-4 Standards

Status Indicators

External Alarm Output

Stainless Steel Enclosure

Facility Monitoring Systems Interface

Versatile Mounting Options

External Vacuum Source
